Q.Cafe is a modern, interactive website for a coffee shop that allows customers to order coffee based on their mood, anytime and anywhere. The website features a clean and user-friendly design, using HTML, CSS, and JavaScript to deliver a responsive and enjoyable experience for coffee lovers.

## Features

- Browse coffee options based on your mood
- Place orders for delivery or pick-up
- View location and shop timings
- Interactive elements built with JavaScript
- Fully responsive layout

## Technologies Used

- **HTML**: Structuring the web content
- **CSS**: Styling the website and making it responsive
- **JavaScript**: For interactive elements and dynamic content
